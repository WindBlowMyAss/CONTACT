## OPENPOSE算法输出骨骼

#### `GET /api/pose/bone`

> 获取骨骼关键点数据

> 参数：无

+ data：

    |       |类型   |备注   |
    | ----- | ----- | ---- |
    | bones | `list` |      |

    + bones元素

        |       |类型   |备注                                                     |
        | ----- | ----- | ------------------------------------------------------- |
        | id    | `int` | 区分场景中多个目标的标志位                                |
        | type  | `int` | 关键点序号，对应不同部位，共25个                           |
        | value | `list`| 关键点数值，为一个三元组，包含了空间平面的位置坐标以及置信度c|

+ 示例

```
GET /api/pose/bone

  {
  	"message": "OK",
  	"timestamp": 123456789012,
  	"data": {
  		"bones": [
  			{
                  "id": -1,
                  "type": 0,
                  "value": [1503.68,479.263,0.115289],
  			},
  			// ...
  		]
  	}
  }
  ```
